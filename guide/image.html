Image

A picture is worth a thousand words!

Basic

A basic image is created with the use of img tag. The img tag is an inline element like a sentence in a p tag.

Figure

To make an image into a block-level element, wrap the img tag inside a figure tag. The figure tag is an block-level elements that will allow additional visual effects and elements such as adding a border, adding a shadow, and adding a caption.

Caption

To add a caption to an image, the img tag must be wrapped inside a figure tag and with the caption texts wrapped inside a figcaption tag. The image caption will be displayed right under the image.

Bordered

To add a border around an image, the img tag must be wrapped inside a figure tag. The figure tag must have a bordered class applied. For example, <figure class="bordered">...</figure>.

Shadowed

To add a shadow around an image, the img tag must be wrapped inside a figure tag. The figure tag must have a shadowed class applied. For example, <figure class="shadowed">...</figure>.

A figure tag can has both the bordered and shadowed classes applied.
For example: <figure class="bordered shadowed">...</figure>.

Scale / Size

By default, the image in the figure tag will be displayed in its original resolution or size. If the image is too big, additional classes are available to scale or size the image down from its original size: x-small (25%), small (30%), and medium (50%).

For example: <figure class="small">...</figure>

Alignments

Alignments can be applied to img and figure elements. When applying alignment to figure element, do not apply the alignment class again to the img element.
