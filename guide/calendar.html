Calendar

A simple calendar to organize important dates.

See Demo

Notes on Accessibility
The calendar grid view is screen reader friendly but not ideal. It is recommended to switch to list view for screen reading.

The calendar component is built on top of the tab component. To start, create a div element with the with-tabs class. Append the calendar class to the class attribute.

<div class="with-tabs calendar" role="application">
</div>

Inside the div element, create an unordered list (ul) element with the tabs class. This unordered list will hold the months, which will be specified next.

<div class="with-tabs calendar" role="application">
    <ul class="tabs" role="tablist">
    </ul>
</div>

To specify the months for the calendar, create list (li) elements inside the ul element. Each li element will need an attribute called data-month. The data-month attribute will hold the numerical month value. For example: 1 is January, 2 is February, 3 is March, and so on.

By default, the year of the month is always the current year. To specify a month in a past or future year, add the data-year attribute to the li element with the value of the desired four-digit year.

Tip: the data-year attribute is needed only when specifying a month in a past or future year. If the month is for the current year, the data-year attribute is not needed.

Warning: while you can specify the year of the month, you can also do them in any order that you desire. However, when constructing the calendar, it is highly recommended to build in sequential order.

<div class="with-tabs calendar" role="application">
    <ul class="tabs" role="tablist">
        <li data-month="12" data-year="2013" role="presentation">
        </li>
        <li data-month="1" role="presentation">
        </li>
    </ul>
</div>

The next step is to create the days for the month. To begin, create another unordered list (ul) element with the days class inside the month li element.

<div class="with-tabs calendar" role="application">
    <ul class="tabs" role="tablist">
        <li data-month="12" data-year="2013" role="presentation">
            <ul class="days">
            </ul>
        </li>
        <li data-month="1" role="presentation">
            <ul class="days">
            </ul>
        </li>
    </ul>
</div>

Inside the newly created ul element, create list (li) elements to represent the days that have agendas. Days without agendas do not need to be specified. Each li element must have the data-day attribute with numerical value between 1 and 31. The numerical value specifies the day of the month that will have an agenda.

<div class="with-tabs calendar" role="application">
    <ul class="tabs" role="tablist">
        <li data-month="12" data-year="2013" role="presentation">
            <ul class="days">
                <li data-day="8">
                </li>
            </ul>
        </li>
        <li data-month="1" role="presentation">
            <ul class="days">
                <li data-day="4">
                </li>
            </ul>
        </li>
    </ul>
</div>

To specify the details of the agenda for a particular day, create two span elements inside the day li element. One span must have the class of title, and the other one must have the class of desc. The title span element holds the title. The desc (stands for description) span element holds the description of the agenda.

To change the color of the agenda items on the calendar, add the data-cat attribute to the title span element. By default, the color of the agenda item is gray. The acceptable data-cat values are info (blue), success (green), warning (yellow), danger (red), and holiday (purple).

Tip: you can also use the color name (blue, green, yellow, red, and purple) for the data-cat value.

<div class="with-tabs calendar" role="application">
    <ul class="tabs" role="tablist">
        <li data-month="12" data-year="2013" role="presentation">
            <ul class="days">
                <li data-day="8">
                    <span class="title" data-cat="warning">Title</span>
                    <span class="desc">Lorem ipsum dolor sit amet, consectetur
                    adipisicing elit.</span>
                    </li>
                </ul> <!-- end days ul -->
        </li> <!-- end month li -->
        <li data-month="1" role="presentation">
            <ul class="days">
                <li data-day="4">
                    <span class="title" data-cat="danger">Title!</span>
                    <span class="desc">
                        <p><strong>Hello world!</strong></p>
                    </span>
                </li>
            </ul> <!-- end days ul -->
        </li> <!-- end month li -->
    </ul> <!-- end tabs ul -->
</div> <!-- end calendar div -->
