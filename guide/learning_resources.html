Learning Resources

A simple and organized way to manage learning (or any) resources.

Learning resources are essentially a glorified and interactive way of listing resources. A learning resource can be a book, video, tool, audio, website, or document. Each learning resource contains an image (if available), a title, an author or a source, tags, and additional content to be shown when clicked.

To create a collection of learning resources, create a <div> element with the class of with-learning-resources. This <div> element will serve as the wrapper for the individual resources. To create a resource item. create a <div> element with class of resource follow the type of the resource: book, video, audio, tool, website, or document.

Inside the resource <div> element, an image can be displayed by creating a <div> element with the class of image and using the img element to link to your image. The size of the image is recommended at and will be scaled to 100 by 150 pixels. If no img element is specified, an icon representing the resource will be displayed.

If the image is not available or not needed, the <div> element with the class of image must still be specified.

After the image, a title, source or author, and tags can be listed. To start, create <div> element with class of cover-info. This <div> element will serve as a wrapper to hold the title, source, and tags. The title can be displayed by creating a <div> element with the class of title. The author or source can be displayed by creating a <div> element with the class of author-source The tags can be displayed by creating a <ul> element with the class of meta.

The maximum line of title and author/source is two. If more than two lines, they will be truncated. The tags are specified by using a list element, so each tag is a li element.

The last part of a learning resource is the additional or detailed content to be displayed after a click or touch (if on touch devices). This part is wrapped inside a <div> element with the class of expanded-info. Inside this wrapper you can create individual <div> element with the classes of notes and actions. The notes <div> element holds whatever the content needs to be. The actions <ul> holds a list of link/button to external resources.

For headings, h4 is recommended. Also, do not forget to add the clearfix helper after the outer most wrapper.

You can create a learning resource collection with just one resource; however, it is not recommended.
