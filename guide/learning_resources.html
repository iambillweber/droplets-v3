Learning Resources

A simple and organized way to manage learning (or any) resources.

See Demo

Learning resources are essentially a glorified and interactive way of listing resources. A learning resource can be a book, video, tool, audio, website, or document. Each learning resource contains an image (if available), a title, an author or a source, tags, and additional content to be shown when clicked.

Creating a Learning Resource

To create a collection of learning resources, create a <div> element with the class of with-learning-resources. This <div> element will serve as the wrapper for the individual resources. To create a resource item, create a <div> element with class of resource follow by the type of the resource: book, video, audio, tool, website, or document.

<div class="with-learning-resources">
    <div class="resource book">
    </div>
</div>

Inside the resource <div> element, an image can be displayed by creating a <div> element with the class of image and using the img element to link to your image. The size of the image is recommended at and will be scaled to 100 by 150 pixels. If no img element is specified, an icon representing the resource will be displayed.

If the image is not available or not needed, the <div> element with the class of image must still be specified.

<div class="with-learning-resources">
    <div class="resource book">
        <div class="image">
            <img src="path_to_img.jpg" alt="describe your image" />
        </div>
    </div>
</div>

After the image, a title, source or author, and tags can be listed. To start, create <div> element with class of cover-info. This <div> element will serve as a wrapper to hold the title, source, and tags. The title can be displayed by creating a <div> element with the class of title. The author or source can be displayed by creating a <div> element with the class of author-source The tags can be displayed by creating a <ul> element with the class of meta.

Pleas limit title and author/source to two lines. Two lines are more than enough for a proper long title or source.

<div class="with-learning-resources">
    <div class="resource book">
        <div class="image">
            <img src="path_to_img.jpg" alt="describe your image" />
        </div>
        <div class="cover-info">
            <div class="title"></div>
            <div class="author-source"></div>
            <ul class="meta">
                <li>Tag 1</li>
                <li>Tag 2</li>
                <li>Tag 3</li>
            </ul>
        </div>
    </div>
</div>

The last part of a learning resource is the additional or detailed content to be displayed after a click. This part is wrapped inside a <div> element with the class of expanded-info. Inside this wrapper you need to create a <div> element with the class of notes and an <ul> element with class of actions respectively. The notes <div> element holds whatever the content needs to be. The actions <ul> holds a list of buttons to external resources. Do not forget the clearfix at the end!

<div class="with-learning-resources">
    <div class="resource book">
        <div class="image">
            <img src="path_to_img.jpg" alt="describe your image" />
        </div>
        <div class="cover-info">
            <div class="title"></div>
            <div class="author-source"></div>
            <ul class="meta">
                <li>Tag 1</li>
                <li>Tag 2</li>
                <li>Tag 3</li>
            </ul>
        </div>
        <div class="expanded-info">
            <div class="notes"> ... </div>
            <ul class="actions">
                <li><a class="btn info" href="#">Action 1</a></li>
                <li><a class="btn success" href="#">Action 2</a></li>
                <li><a class="btn danger" href="#">Action 3</a></li>
            </ul>
        </div>
    </div>
</div>
<div class="clearfix"></div>

The color of the buttons can be changed. Refer to the buttons in the guide.

To add another or more resources, simply append another <div> element with classes of resource and the type of the resource.

<div class="with-learning-resources">
    <div class="resource book">
    </div>
    <div class="resource video">
    </div>
    <div class="resource website">
    </div>
</div>

Typography in the Notes

The notes <div> element under expanded-info <div> element accept whatever HTML contents. Thus, following the proper typography is essential. Below is a list of suggestions to follow:

  • Use <h4> element for headings.
  • Use <p> element for body copy.
  • Do not use Tabs component.
  • Do not use Accordions component.
