Internal Page Navigation

An internal page navigation for long content pages using anchor tags.

To use a sub navigation for long content pages, add the with-subnav class to the <div> element. Any <h2> element inside the <div> element will be used by the JavaScript to create the sub menu box. The JavaScript will automatically anchor each menu item to the contents.

Notes: the <h2> elements are required for JavaScript. They will be used to create the sub menu navigation.
