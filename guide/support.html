Support

If you are experiencing issues implementing the HTML, CSS, and JavaScript, please follow the procedure stated below. Level 3 VIP passes are restricted to few individuals.

Level 1

This is the initial support level responsible for handling basic questions and issues stemming from instructional designers. Designers well-versed in writing and creating content pages using the HTML, CSS, and JavaScript framework will provide level 1 support to their colleagues. How that support structure is designed and executed is up to the Design and Development team.

FYI: Whether if level 1 support is available or not, it is always a good idea to check out the Issues page on Media Team GitHub account. Your issue or question might already been reported or answered.

Level 2

This is a more in-depth technical support level that often involves complexity and intricacy in the HTML creation. If level 1 support is insufficient, please contact Bryan Bortz via email to set up a 1:1 consultation and provide the information listed below:

  • Code snippet
  • Screen shot of display issues (if available)
  • Course information (if available)

During the consultation, Bryan will work directly with the designer to resolve their issues and/or questions.

Level 3

This is the highest level of support in our three-tiered technical support model responsible for handling the most advanced problems. Problems should rarely escalate to level 3 support but if they do, Bryan will work directly with Ethan to layout the problem and discuss possible solutions. This will be transparent to designers and is in place to give Bryan an outlet for his inquiries.

Bug Reporting

If you discover a bug please report it to a level 1 support person to confirm the issue. If it is indeed a bug, please report the issue on our Media Team GitHub account. Ethan monitors our account on a regular basis and is automatically alerted when bugs are reported.
