FEATURE REQUESTS

If you have a feature request (and you’ve already verified that your Style Guide doesn’t address the issue) please fill out the Droplets Feature Request form. Suggestions that fall under the umbrella of a feature request include:

  • Enhancements to existing tools (visual or functional)
  • Additional options for visually designing pages
  • New tools to make pages interactive

Submitted requests are automatically sent to the Feature Request group for further review.
