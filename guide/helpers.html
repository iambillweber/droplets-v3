Helpers & Accessibility

Miscellaneous classes and accessibility markups to help enhance the page content and screen reader friendly.

HTML / CSS Helpers

Clearfix

Clear CSS float property on any elements. Usually used with columned layouts and floating elements.

<div class="clearfix"></div>

Accessibility

When in doubt, use role="presentation".

Websites and web applications use Accessible Rich Internet Applications Suite (WAI-ARIA) to define a way to make web content more accessible to people with disabilities. It is a standard developed by the World Wide Web Consortium (W3C), an international community where private and public organization work together to develop web standards.

Throughout Droplets, element and component code examples already have ARIA markups included. Please pay attention to the notes in regard to each element or component if available. If for any reasons that you need different markups other than the ones provided or want to learn more about WAI-ARIA, see resource links below.

  • Mozilla Developer Network (MDN) ARIA
  • A List Apart: The Accessibility of WAI-ARIA
  • Introduction to WAI ARIA
  • Using WAI-ARIA Landmarks

Although WAI-ARIA is currently still in development process, modern web browsers and screen reader software are already supporting most of the essential WAI-ARIA standards. For best supports, use the latest stable version of Apple Safari web browser.
