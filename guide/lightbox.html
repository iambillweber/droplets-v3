Lightbox

A way to display high resolution images.

Lightbox only works with img or figure tag. To use lightbox, add the with-lightbox class to a img or figure tag. You can also applied lightbox effect to a collection of images rather than individual image.

Basic

To create a basic lightbox effect for an image, use the img tag and to add caption to the lightbox effect, use the alt attribute of the img tag. The caption will not be display in the unexpanded image for a basic lightbox setup.

Caption

To display caption in both unexpanded and expanded view of the image, use the figure tag with figcaption. The alt attribute of the img tag will be ignored. The figcaption must be used when using a figure tag for lightbox effect.

Collection

When there are multiple images and needs to in a grid layout, use a div tag with the with-lightbox class. Inside the div tag, either use img or figure tag. It cannot be both.

Alignments

A single lightbox image can be aligned left or right by appending the left or right class. Images or figures in a collection does not need alignment classes.
